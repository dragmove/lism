## About

- A Bundle of Typescript util Library to help developers.

## Install

```sh
npm install lism --save
yarn add lism
```

## Run Tests

```sh
npm run test
```

## Build documents

Build typdoc documents to `docs` path

```sh
npm run docs
```

### Update Release Version

```sh
npm run version
```
